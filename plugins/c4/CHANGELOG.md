# Changelog

## [0.1.0](https://github.com/panbanda/islands/releases/tag/c4-plugin-v0.1.0) (2025-01-08)

### Features

* Initial release of C4 Model Generator plugin
* Procedural 8-phase workflow for analyzing codebases
* Support for documenting single repos or multiple services
* Templates for all C4 element types (persons, systems, containers, components, relationships, flows, deployments)
* Reference documentation for C4 schema and common patterns
* Scripts for analyzing Rust codebases and validating YAML output
